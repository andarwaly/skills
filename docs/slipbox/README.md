# Slipbox Docs

Human-facing documentation for the slipbox skill family.

- [setup-slipbox](./setup-slipbox.md) — One-time onboarding: discovers vault conventions, analyzes writing style, initializes idea.db
- [clip-resource](./clip-resource.md) — Fetch a URL and write it as a frozen Resource, for users without a clipper tool
- [surface-ideas](./surface-ideas.md) — Surface 5-10 discussion-worthy candidate ideas from a clipped Resource, checking diversity across multiple dimensions
- [write-literature-note](./write-literature-note.md) — Turn a candidate idea into a finished, single-claim literature note via Socratic discussion
- [write-reference-note](./write-reference-note.md) — Manually-triggered definitional note for a recurring term, extendable across sessions
- [write-evergreen-note](./write-evergreen-note.md) — Connect two or more existing notes into a new, purely original idea via Socratic discussion
