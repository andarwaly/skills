# Slipbox Docs

Human-facing documentation for the slipbox skill family.

- [setup-vault](./setup-vault.md) — One-time onboarding: discovers vault conventions and writing style
- [clip-resource](./clip-resource.md) — Fetch a URL and write it as a frozen Resource, for users without a clipper tool
- [surface-ideas](./surface-ideas.md) — Surface 5-10 discussion-worthy candidate ideas from a clipped Resource
- [discuss-idea](./discuss-idea.md) — Socratic conversation from a candidate idea to a finished literature note
- [write-reference-note](./write-reference-note.md) — Manually-triggered definitional note for a recurring term or source
- [discuss-connection](./discuss-connection.md) — Connect two or more existing notes into a new, purely original idea
