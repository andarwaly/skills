# Skills

Skills are grouped into domain buckets for discoverability and maintenance.

## Buckets

| Bucket | Purpose |
|---|---|
| [obsidian](./obsidian/) | Obsidian plugin integration, templates, vault automation |
| [figma](./figma/) | Figma plugin design, development, and workflow automation |
| [productivity](./productivity/) | General productivity and workflow skills |
| [misc](./misc/) | Uncategorized or single-shot skills |

## Layout

```
skills/
├── <bucket>/
│   ├── <skill-name>/    # skill directory (SKILL.md + assets)
│   └── README.md         # bucket index with linked skill list
└── README.md             # this file
```

Each bucket's `README.md` lists its skills with a one-line description and links to their directories.
